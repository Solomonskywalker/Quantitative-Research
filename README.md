# Quantitative Pattern Trading Research

## Overview

This repository contains quantitative research and analysis of financial markets, focusing on pattern-based trading strategies and risk-adjusted performance evaluation.

The research examines historical market data from multiple asset classes, including:

- SPY (S&P 500 ETF)
- Gold
- EUR/USD Foreign Exchange Market

The objective is to study market behavior, evaluate trading opportunities, and measure performance using quantitative finance metrics.

## Key Metrics Studied

### Sharpe Ratio

Measures risk-adjusted returns by comparing excess returns to portfolio volatility.

### Maximum Drawdown

Measures the largest peak-to-trough decline experienced during the investment period.

### Returns Analysis

Evaluates cumulative and periodic returns to assess performance.
